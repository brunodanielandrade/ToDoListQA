# ToDoList - Java

## Descrição

Aplicação simples em Java que simula um CRUD de tarefas.

## Funcionalidades

* Adicionar tarefa
* Listar tarefas
* Remover tarefa

## Regras de Negócio Testadas

* Uma tarefa deve ser adicionada corretamente
* Não é permitido adicionar tarefa com descrição vazia
* Não é permitido adicionar tarefa nula

## Testes Unitários

* Teste de sucesso: adiciona uma tarefa válida
* Teste de erro: tenta adicionar tarefa vazia ou nula e gera exceção

## Tecnologias

* Java
* JUnit 4

## Execução

Executar a classe `TodoServiceTest` para rodar os testes.
